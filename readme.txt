Project created by Taylor Poulos winter 2012

A simple game of tron with networking and AI

Networking assisted by PodSixNet (see tron_client.py for further information)

To install and run:
*Remove tron from zip file
*Install pygame and PodSixNet (look in tron_client.py for URL)
*run RUN_txTRON.py

To run networking:
*run tron_server
*change config.SERVER_IP to the IP of the server (or "localhost" if on the same machine as the server)
*Run tron.py
*enter networking tab
	Enter networking tab on another computer to reserve that player's spot
	Press spacebar to add as many AI as desired
	Press enter on any computer to run
